# Nested Dictionary
*** nestes dictionary is used everywhere around ***
**ApI**
**Json**
**Ai-Resposes**
**Data-Base**
**Web-Development**
# examples
```python

student = {
    "gabriel": {
        "age": 23,
        "language": "python"
    }
}
```

# we also have dictionary that contains list.
**example**
```py
student = {
    "language": [
        "go-lang",
        "python",
        "git"
    ]
}
```
# we also have list that contains dictionary
```py
student = [
    {"name": "Gabriel"},
    {"name": "JohnDoe"}
]
```
**this is probably the most common data structure in backend Development.**
